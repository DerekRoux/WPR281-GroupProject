# Bug Tracking System - Group Project

## Project Overview
This is a web-based system designed to track software issues, assigned to specific team members across different projects.

## Roles & Responsibilities
- **Lead Developer:** JavaScript Logic & localStorage Integration.
- **UI/UX Designer:** Bootstrap Interface & Dashboard.
- **Data & QA Specialist:** Data Architecture, Static Entities, and Test Cases.
- **Project Manager:** GitHub Management & Final Review.

## Technical Details
- **Persistence:** Uses `localStorage` to save bug data.
- **Framework:** Built with HTML5, CSS3, and Bootstrap 5.
- **Data Structure:** Objects for People, Projects, and Bugs.
